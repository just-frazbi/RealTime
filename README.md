## 🇪🇳 Description
# RealTime — a plugin that synchronizes in-game time with real-life time ⏰

RealTime automatically syncs Minecraft time with your real-world time, making the game more immersive and realistic.

Perfect for all server types — from survival to roleplay, where atmosphere matters.


**🔧 Main Features:**
- Automatic synchronization of in-game time with real time
  
- Simple control via commands
  
- Flexible configuration:
  choose specific worlds
  set update interval
  
- Permissions system
  
- Easy configuration via config.yml



**📜 Commands:**

- /realtime enable — enable synchronization
  
- /realtime disable — disable
  
- /realtime reload — reload config
  
- /realtime status — show current status



## 🇷🇺 Описание
# RealTime — плагин для синхронизации игрового времени с реальным временем ⏰

RealTime автоматически синхронизирует время в Minecraft с вашим реальным временем, делая игровой мир более живым и реалистичным.

Плагин отлично подходит для любых серверов — от выживания до RP, где важна атмосфера и реализм.


**🔧 Основные возможности:**
- Автоматическая синхронизация игрового времени с реальным
- Простое управление через команды
- Гибкая настройка:
  выбор конкретных миров
  настройка интервала обновления
- Система прав (Permissions)
- Лёгкая настройка через config.yml



**📜 Команды:**
- /realtime enable — включить синхронизацию
  
- /realtime disable — выключить
  
- /realtime reload — перезагрузить конфиг
  
- /realtime status — показать текущий статус
